### What I'm working on

My **Final Degree Project**, supervised by Dr. Emilio Sánchez Tapia at CEIT's robotics lab, focuses on **multi-robot fleet management** using the [Open-RMF](https://open-rmf.org/) framework on top of ROS 2 Jazzy. I've built a full task management stack — Docker-based API server, Fleet Manager Core, Fleet Adapter with JWT authentication, and a Node-RED HMI for real-time monitoring — validated both in Gazebo simulation and on physical robots (Robotnik RB-Vogui+, Omron LD90, Husarion ROSbot).

📌 **Main repo:** [fleet-manager-hmi](https://github.com/Gparot/fleet-manager-hmi)
